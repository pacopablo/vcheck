vCheck provides a Python package for interacting with and managing vCenter via Python objects. The API is modeled after the main objects in a vCenter deployment. Namely:

Datacenter:

Models the datacenter. Allows operations on various stuff

Host:

Host based operations such as networking, datastores, user management, etc.

VM:

VM operations such as start/stop/provision.
