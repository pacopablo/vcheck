Connection¶

class vmware.connection.VimConnection(url, username, password)¶

When instantiating a VimConnection object, it requires a URL to the vCenter or ESXi web service. The web services is usually located at

https://<vcenter_or_esxi_host>/sdk

The username and password are also required. Upon instantiation, the VimConnection object will log into the vCenter or ESXi host. All other classes use a VimConnection object to interact with vCenter and ESXi hosts.

connection(read-only)¶

VMWare.Vim.VimClient managed object. If the VimConnection was unable to connect, this will be None. Classes must use vmware.connection.VimConnection.connection as the connection to the vCenter or ESXi host.
